# Simulated Pilot Factory Profile

## Purpose

This document defines the representative factory context used for the HumanEnerDIA simulated pilot benchmark.

The profile was selected to match the intended WASABI pilot-factory scope while maintaining a clear distinction between a simulated benchmark and a completed live factory deployment.

## Scope Boundary

This profile represents a realistic simulated Romanian manufacturing context. It is not presented as evidence of a completed live deployment at a Romanian host factory.
